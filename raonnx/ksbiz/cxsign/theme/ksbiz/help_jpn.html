브라우저 인증서 이용안내

브라우저 인증서란?

글로벌 웹표준(HTML5)를 지원하는 브라우저의 웹저장소 영역에 인증서를 저장하여 사용하는 방식으로, 공인인증 프로그램을 설치할 필요가 없는 HTML5기반의 공인인증서입니다.

브라우저 인증서 발급/복사 방법

  • 신규/재발급할 경우
    1. 01. 공인인증센터 발급화면에서 브라우저 인증서 방식 선택
    2. 02. 인증서 발급절차에 준하여 인증서를 발급받으면, “브라우저”영역에 저장
  • 기존 발급받은 인증서를 이용할 경우
    1. 01. 로그인 창에 접속 후 “브라우저 인증서로그인”클릭
    2. 02. “인증서찾기”선택 후 “인증서변환프로그램 다운로드” 설치

    3. 03. 기존에 발급받은 인증서를 선택하고, “.pfx”화일로 저장

    4. 04. “인증서찾기”에서 “인증서(*.pfx, *p12)열기”선택하여 저장된 파일 선택한 후 인증서 암호 입력

    5. 05. 인증서암호 입력 후 “브라우저”영역에 인증서가 저장됨

  • 다른 이용방법
    1. 01. 당행홈페이지 > 공인인증센터 > 공인인증서 > 인증서관리 > 인증서 관리실행 선택
    2. 02. 해당 인증서를 선택한 후 인증서 내보내기하여, “ pfx”화일로 저장
    3. 03. 로그인 창에 접속 후 “브라우저 인증서로그인”클릭
    4. 04. “인증서찾기”에서 “인증서(*.pfx, *p12)열기”선택하여 저장된 파일 선택
    5. 05. 인증서암호를 입력하면 “브라우저”영역에 인증서가 저장됨

이용시 유의사항

  • 웹 브라우저에 저장된 공인인증서는 당행에서만 사용 가능합니다.(타행은 미지원)
  • 웹 브라우저내에 저장되므로 타 브라우저간에 연동(호환)되지 않습니다.
  • 웹 브라우저 초기화시에는 인증서가 삭제됩니다.
